# On the Wall

A single-page art gallery website built as a fellowship exercise in core CSS layout concepts.

**DevWeekend Fellowship — Task**

## What this demonstrates
- Multi-section static layout using **flexbox**
- **Absolute vs relative** positioning (museum-plaque effect on each painting frame)
- The CSS **box model** — margin, padding, border used deliberately throughout

## Pages
- `index.html` — home page with a gallery of 5 famous paintings, each with a short note on its meaning
- `about.html` — about the project and the artists featured

## Tech
Plain HTML, CSS, and Google Fonts (Fraunces + Inter). No frameworks, no build step.

## Running locally
Open `index.html` directly in a browser, or use VS Code's Live Server extension for auto-reload while editing.
